# Adobe Target Floodgate entdecken {#get-started}


## Was ist Floodgate?

Floodgate ist ein Funktions-Management- und Experimentiersystem, das Anwendungen dabei hilft, neue Erlebnisse sicher, schrittweise und mit weniger Risiko bereitzustellen.

Anstatt eine neue Funktion für alle gleichzeitig zu aktivieren, ermöglicht Floodgate Produktteams:

* Funktionen schrittweise veröffentlichen
Beginnen Sie mit einem kleinen Prozentsatz an Benutzenden und erhöhen Sie dann im Laufe der Zeit. Wenn sich etwas nicht wie erwartet verhält, kann die Funktion schnell deaktiviert werden - ohne Ausfallzeiten oder Neuinstallation von Software.

* Targeting von Funktionen für bestimmte Zielgruppen
Aktivieren Sie neue Funktionen für ausgewählte Gruppen (z. B. Beta-Programme, bestimmte Regionen oder bestimmte Pläne), damit Änderungen validiert und verfeinert werden können, bevor sie allgemein verfügbar sind.

* Durchführen von Experimenten und A/B-Tests
Probieren Sie verschiedene Versionen einer Funktion aus (z. B. alternative Layouts, Flüsse oder Empfehlungen) und messen Sie, welche besser funktioniert, indem Sie Daten verwenden, um Produktentscheidungen zu leiten.

* Geringeres Risiko für kritische Versionen
Große Änderungen können in kleinere, kontrollierte Rollouts unterteilt werden, sodass Probleme leichter erkannt, verstanden und behoben werden können, bevor sie alle betreffen.

* Schnelles Reagieren auf Probleme
Wenn ein Fehler oder ein Leistungsproblem auftritt, können Teams nur die betroffene Funktion deaktivieren, sodass der Rest der Anwendung stabil bleibt.

In der Praxis hilft Floodgate Endbenutzern dabei, häufiger Verbesserungen mit weniger Unterbrechungen zu erleben: Neue Funktionen können früher eintreffen, sorgfältiger getestet werden und schnell auf der Grundlage von echtem Feedback angepasst werden.
