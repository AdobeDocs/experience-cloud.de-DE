# Web-Dienste {#web-services}

Backend-Services integrieren mit Flags mithilfe einer Server-seitigen SDK. Wählen Sie die SDK aus, die Ihrem Technologie-Stack entspricht.

## Verfügbare SDKs {#available-sdks}

>[!NOTE]
>
>Die Server-seitige SDK-Dokumentation wird derzeit vorbereitet und wird in Kürze verfügbar sein. Wenden Sie sich an Ihren Adobe-Support-Mitarbeiter, um frühzeitig Zugang zu erhalten.

## Other languages {#other-languages}

Wenden Sie sich an den Flags-Support , um Anleitungen zur Integration mit Ihrem spezifischen Stack zu erhalten.
